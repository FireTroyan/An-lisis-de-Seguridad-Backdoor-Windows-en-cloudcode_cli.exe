# Anlisis-de-Seguridad-Backdoor-Windows-en-cloudcode_cli.exe
Soy un investigador de seguridad independiente y víctima directa de una brecha de seguridad masiva que comprometió mi privacidad e identidad digital. Durante meses he documentado exhaustivamente una puerta trasera (backdoor) de nivel APT en sistemas Windows que ha sido ignorada por Microsoft a pesar de haberles proporcionado evidencia técnica.

¿Qué voy a informar?

Este repositorio contiene el análisis forense completo de la backdoor cloudcode_cli.exe, que incluye:

    Evidencia de persistencia multinivel: Servicios, Registry, COM Hijacking

    Certificado DigiCert comprometido: Firmas digitales legítimas utilizadas para evadir detección

    Comunicaciones C&C: 57 direcciones IP externas y patrones de exfiltración

    Técnicas APT avanzadas: Ofuscación multicapa (PGP, RAR, 7Z) y anti-forense

    IOCs completos: Hashes, patrones de red, artefactos de persistencia

¿Por qué lo hago?

Microsoft recibió múltiples reportes a través de sus canales oficiales, asignó IDs de vulnerabilidad, pero trató el caso como un "bug común" susceptible de ser parcheado, ignorando la naturaleza sistémica del compromiso. Fui censurado y mi identidad digital fue vulnerada por pensar diferente y exigir transparencia.
Propósito

    Alertar a la comunidad global sobre una vulnerabilidad real que afecta a millones de usuarios de Windows

    Proporcionar a equipos CERT/CSIRT los indicadores técnicos necesarios para detectar y remediar este compromiso

    Exigir responsabilidad corporativa mediante transparencia técnica irrefutable

    Proteger a otros posibles afectados que, como yo, han perdido su privacidad sin saberlo

Estructura del repositorio

    /evidencia_tecnica - Análisis forense completo y capturas

    /iocs - Indicadores de compromiso para equipos de seguridad

    /persistencia - Mecanismos de persistencia identificados

    /comunicaciones_c2 - Análisis de tráfico y patrones C&C

Este no es un ataque, es una alerta de seguridad de interés público. La verdad técnica debe prevalecer sobre el silenciamiento corporativo.
